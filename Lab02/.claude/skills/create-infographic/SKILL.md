# Lab Infographic Creator

When creating infographics from lab documentation:

1. Read the input markdown file specified by the user
2. Extract key data points, statistics, and facts:
   - Attacker profiles and characteristics
   - Attack statistics (data leaked, victims, etc.)
   - Timelines of events
   - Key achievements or incidents

3. Generate an interactive HTML infographic with:
   - Visual cards for each attacker/entity
   - Statistics displayed prominently with icons
   - Timeline visualization of events
   - Color-coded severity indicators
   - Comparison charts between different attackers
   - Visual hierarchy showing relationships

4. Include professional styling for:
   - Modern card-based layout
   - Eye-catching typography for statistics
   - Icon-enhanced data points
   - Responsive grid system
   - Dark theme suitable for cybersecurity content
   - Print-friendly version

5. Use the generator.py script to automate the visualization process
6. Output the HTML file to the project directory

The infographic should be visually engaging, data-driven, and suitable for presenting security research findings.
